Git Bash中alias的使用

  • 在我们安装完 git 之后，右键会有 Git Bash 命令行工具

  • 当你在使用命令行时，有没有发现命令行好长，一不小心就敲错了几个字母？下面教你如何使用 alias 合理装逼

  1. 右键打开 Git Bash 命令行工具，并执行用 vim 打开 aliases.sh 文件

  1. 如果你操作过 aliases.sh 此时可能提示你以何种方式打开，我们选择“直接编辑（（E））”（如果你以前没有操作过 aliases.sh 这一步可能不会出现）

  2. 此时你已经进入 aliases.sh 文件，点击 i 或者 a 开始编辑，此时左下角会出现“–插入–”或者“–INSERT–”的提示

  3. aliases.sh 文件中已经有两个 alias

你可以添加你需要的

  1. 编辑完成之后，点击 Esc，输入:wq，或者点击 Esc 之后，输入两次大写 Z，即保存退出

  2. 关闭 Git Bash，重新打开 Git Bash，此时如果你想要在当前文件夹打开 vscode 可以输入
